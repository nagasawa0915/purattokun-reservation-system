## 🚀 パーセンテージ統一決定の経緯

### 背景となった問題
2025年1月28日時点で以下の座標系問題が発生していました：

1. **ウィンドウリサイズでの位置ズレ**: viewport units (vw/vh) が不安定
2. **複数座標系の混在**: ピクセル・パーセント・viewport unitsが混在
3. **レスポンシブ対応困難**: 固定値による制約
4. **保守性の低下**: 複雑な座標ルールによる理解困難

### 解決方針の決定（2025年1月29日）
**ユーザーからの明確な要求**: 「ウィンドウサイズ変更でもキャラクターと背景が完全に同期する」

**技術的検証結果**:
- viewport units (vw/vh): 親要素のサイズ変更に追従しない問題確認
- 固定ピクセル値: レスポンシブ対応不可能
- **パーセンテージ**: 親要素基準で自動調整、最も安定した動作

### 実装決定内容
**座標系統一方針**: パーセンテージ（%）による完全統一
**レイヤー設計**: 2レイヤー構成による明確な分離

## 🏗️ 2レイヤー設計の技術的根拠

### レイヤー構成

#### 1. HTMLレイヤー（設定管理）
```html
<div id="purattokun-config" style="display: none;"
     data-x="35"            <!-- 横位置：35%（パーセンテージ値） -->
     data-y="75"            <!-- 縦位置：75%（パーセンテージ値） -->
</div>
```

**役割**:
- 座標値の一元管理
- HTMLから直接変更可能な設定インターフェース
- JavaScriptによる動的読み込み対応

#### 2. CSSレイヤー（表示制御）
```css
#purattokun-canvas {
    position: absolute;
    left: 35%;               /* パーセンテージ使用（確定済み解決策） */
    top: 75%;                /* パーセンテージ使用（確定済み解決策） */
    width: 25%;              /* パーセンテージ使用（確定済み解決策） */
    aspect-ratio: 3/2;       /* 縦横比固定 */
    transform: translate(-50%, -50%);  /* 中心点基準（必須） */
}
```

**役割**:
- 実際の表示位置・サイズ制御
- レスポンシブ対応の自動処理
- 中心点基準による一貫した位置計算

### 2レイヤー設計の利点

1. **関心の分離**: 設定と表示の明確な分離
2. **保守性向上**: 変更箇所の明確化
3. **拡張性確保**: 新しい設定項目の容易な追加
4. **テスト性向上**: 各レイヤー独立でのテスト可能

## 🎯 座標値の設計思想

### 選択された座標値
- **横位置**: 35%（道路付近の自然な位置）
- **縦位置**: 75%（地面レベルの適切な高さ）
- **サイズ**: 25%（背景とのバランスが最適）

### 座標値決定の根拠
1. **ユーザビリティテスト**: 複数の位置での表示確認
2. **背景画像との関係**: 道路・建物との位置関係考慮
3. **レスポンシブ対応**: 各画面サイズでの適切な表示確認
4. **視覚的バランス**: キャラクターの存在感と背景の調和

### aspect-ratio導入の技術的意義
```css
aspect-ratio: 3/2;  /* 縦横比固定 */
```

**導入理由**:
- 画面サイズ変更時の縦横比維持
- キャラクターの潰れ・伸び防止
- 一貫した表示品質の保証

2. **コード簡素化**: 複雑な座標計算処理の削除
3. **保守性向上**: 統一されたルールによる理解しやすさ
4. **性能向上**: 動的計算処理の削減

## 📐 中心点基準システムの継続

### transform基準の維持理由
```css
transform: translate(-50%, -50%);  /* 中心点基準（必須） */
```

**継続する理由**:
1. **位置計算の一貫性**: left/topが中心点位置として機能
2. **既存システムとの互換性**: 編集システムとの整合性維持
3. **直感的な位置指定**: 中心点ベースは位置調整が容易
4. **レスポンシブ対応**: 画面サイズ変更時の安定した動作

### 中心点基準の利点
- 要素の幅・高さに依存しない位置指定
- 均等な拡大縮小時の位置維持
- 回転・変形時の基準点明確化
